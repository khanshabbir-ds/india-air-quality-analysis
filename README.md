🌍 Air Quality Intelligence System (India) 🚀 Overview

A complete data science project analyzing air pollution trends across Indian cities with machine learning and forecasting.

🔥 Key Features

📊 25+ Data Visualizations
📈 AQI Trend Analysis
🤖 Machine Learning Model (Unhealthy AQI Prediction)
📉 Time Series Forecasting (ARIMA)
🌐 Interactive Dashboard (Streamlit)

🛠 Tech Stack

Python
Pandas, NumPy
Matplotlib, Seaborn
Scikit-learn
Statsmodels
Streamlit

▶️ Run Locally

pip install -r requirements.txt
python src/main.py
streamlit run app/app.py

📂 Dataset

India Air Quality Dataset (AQI)
